Secure.ai

Home security powered by AI and Computer Vision Technologies

Facial recognition locks

We present the facial recognition door lock with multi-factor authentication(eg: key, fingerprint, mobile app) in case the face is not recognized. One-time access can be granted by the mobile app for package deliver or other service persons. If the person stays at your property for more than 3 minutes, which is adjustable, the mobile app will also send you an alert.

Multi-location Real-time Video Streaming

View multiple real-time camera video streamings at the same time away from home. Instead of switching from cameras, now you have a centralized app to view all streaming videos by just a few taps.

Secure One-time Access

Allow one-time access to the smart lock via QR-code, backed by cryptographical algorithm. This enables owner to grant temporary access to visitors or tenants. The code is only valid within the specified time period and for one-time use.

Security Social Networking

We present the idea of neighborhoods by grouping users by geo locations. Every neighborhood has a chat room, you can exchange security information with your neighbors. When suspecious activities are detected, you may choose to share the live stream with your neighbors.

Emergency Response

When suspecious activities are captured, you can inform the police using the mobile application and all users in the same neighborhood will be informed.

IFTTT of Home Security

A visual flow-based programming mechanism allows you to create logics and apply them to your security.ai devices:

Example: If suspecious activities are detected, also send the video footage to my own home cloud storage
