超分析师预期

  • 季度revenue 116亿美金
  • Revenue持续下降, 预期也不乐观
  • 新业务增长明显
  • 盘后股票上涨 - 看好公司业务转型

传统业务下滑

Routing -10%
Switching -5%
Data Center -4%

数据中心

  • 业务下降 4%
  • blade server 需求减缓
  • 回归 Rack server

数据中心

  • 继续投入加强 HyperFlex
  • HyperFlex 2016年三月份推出，hyperconverged 系统
  • 客户megascale datacenters, or MSDC超大规模数据中心增多
    • 不一定采纳传统Cisco设备及方案
    • 此类客户追求开放式系统、开源软件等
    • Google, Facebook, BAT等自行开发网络方案

SDDC趋势

  • SDDC: Software Defined Data Center 软件驱动数据中心
  • ACI: Application Centric Infrastructure 应用为中心的基础设施
  • ACI 是Cisco SDDC方案的重要部分, 被证明市场接受
  • 相关业务 +28%。 本季度新加 1,300 Nexus 9000 客户及 450 ACI 客户
  • 软件（与硬件结合）的重要性

Open Architecture

CEO 接受采访时提到会注重 开放架构及生态构建

UCS支持Azure Stack

  • UCS 是思科统一计算平台，提供新一代数据中心解决方案，在2009年首次进入市场
  • 其它支持微软云服务 Azure Stack的厂家包括
    • HPE
    • Dell
    • Lenovo
